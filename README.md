Dev blog source code

- Starter: [Tailwind Nextjs Starter Blog](https://github.com/timlrx/tailwind-nextjs-starter-blog)


## How to start?

- Install: `pnpm install`
  - For Windows, run additionally: `$env:PWD = $(Get-Location).Path`
- Run: `dev`
- Run tests: `pnpm test`
- Debug: `dev`
